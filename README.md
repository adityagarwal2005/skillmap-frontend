# SkillMap — Frontend

React frontend for [SkillMap](https://skillmap-frontend-two.vercel.app) — a hyperlocal skill discovery platform.

**[🌐 Live App](https://skillmap-frontend-two.vercel.app) · [Backend Repo](https://github.com/adityagarwal2005/skillmap-backend)**

## Stack
React 18 · Context API · Axios · Vercel

## Setup
```bash
npm install
echo "REACT_APP_API_URL=http://127.0.0.1:8000" > .env.local
npm start
```
